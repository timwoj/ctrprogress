How It Works

  1. A list of toons for each team is culled from Knate's Master Raid Team List spreadsheet.
  2. We loop through the list of teams, and gather progression data for each toon using the Battle.net API.
  3. Using that toon data, progression for the team is built using the last-killed time for each boss in both Highmaul and Blackrock Foundry. For each boss, if five or more toons have the same last-killed time, then that kill is considered a progression kill for the team as a whole.
  4. This progression data is regenerated every morning at 2AM Arizona time. Prior progression for a team is never erased, only updated, so a team will never go backwards in progression.
  5. Teams are ordered on the display page using the following order: M HC > H HC > N HC. This does mean that teams that have cleared more of N BRF than a team that hasn't cleared any of BRF, even if that team has cleared all of heroic HM. In the case of ties, the groups are listed in alphabetical order
  6. Group Item Level may appear lower than is actually the case. This is due to alternates and bench characters being taken into consideration when overall calculation is made.

If errors are found in the way that progression is calculated, your team has incorrect information displayed, or for any other questions on how this works, contact Tamen on Aerie Peak or @tamenctr on Twitter.
